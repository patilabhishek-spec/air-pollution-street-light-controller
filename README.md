# Air Pollution Based Automatic Street Light Controller

## Description
This project is an IoT-based system designed to control street lights automatically based on air pollution levels and environmental conditions.

## Features
- Real-time air quality monitoring using gas sensor (ESP32)
- Automatic street light intensity control
- Energy-efficient operation
- Wireless data monitoring for smart city applications

## Components Used
- ESP32 Microcontroller
- Gas Sensor
- LEDs / Street Light Model
- Power Supply

## Working
The system monitors air pollution using a gas sensor connected to ESP32. Based on pollution levels and conditions, the street light intensity is automatically adjusted to optimize energy consumption.

## Applications
- Smart Cities
- Energy Efficient Street Lighting
- Environmental Monitoring Systems
